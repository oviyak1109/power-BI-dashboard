# Dashboard Projects

This repository contains two Power BI dashboards built for different datasets:

1. **SuperStore Sales Analysis**
2. **Hospital Patient Analysis**

---

## 1. SuperStore Sales Analysis
**Features:**
- Slicers: Year, Region, State
- KPIs: Sum of Sales, Profit/Loss, Average Sales
- Donut Chart: Sales by Segment
- Pie Chart: Sales by Category
- Bar Charts:
  - Top 10 Profit/Loss
  - Sum of Quantity and Sales by Year & Quarter
  - Profit Margin by Sub-Category

**Folder:** `SuperStore_Analysis`

---

## 2. Hospital Patient Analysis
**Features:**
- KPIs: Number of Patients, Diagnosis Count, Result Count, Number of Doctors
- Donut Charts:
  - Patient Count by Gender
  - Diagnosis Count
- Bar Charts:
  - Patient Count by Year
  - Patient Count by Blood Type
- Slicers: Year, Quarter, State, City, Department, Diagnosis
- Pie Chart: Count of Patients by Age Bins

**Folder:** `Hospital_Analysis`
